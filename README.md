# RSVP Reading App

![example](https://github.com/LKolding/RSVP-app/blob/main/example.gif)

## About
Test and train reading speed, or rapidly digest a long text. Any text, any language... but only Latin alphabet.

This program repeatadly and rapidly prints one word at a time (look below) from a user provided text, centered and conviently placed such that the eyes barely needs to move in order to "scan" the word.
